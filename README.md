# scada-load-forecasting
Short-term (1-hour ahead) industrial load forecasting using SCADA data and ML/DL models.

## Project Structure

```
data/           # SCADA dataset (not uploaded if confidential)
notebooks/      # Experiments and exploration
src/            # Training and forecasting scripts
results/        # Model outputs and evaluation
docs/           # Thesis notes and references
```

Each directory contains a README.md with detailed information about its purpose.
